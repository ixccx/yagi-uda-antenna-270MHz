# Yagi-Uda Antenna Design at 270 MHz

**King Abdulaziz University | EE424 | 2025**

## Overview
Designed and simulated a Yagi-Uda antenna studying the effect 
of parasitic elements on gain and directivity at 270 MHz.

## Results
| Configuration | Gain |
|---|---|
| Dipole only | 2.09 dBi |
| Dipole + Reflector | 7.18 dBi |
| Dipole + Reflector + 4 Directors | 9.26 dBi |
